IP 一旦理性，就是死亡的序章。 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 刘思毅宝宝总已持续日更 1733 天！陪伴中国年轻创业者一起赚钱！一起成长！这两天跟我们 CTO 讨论了一个非常关键的问题：刘思毅做为一个 IP ，为什么非常容易疯狂说错话，为什么总是控制不住自己随意表达？有些时候，虽然我有 IP 经纪人，我有合伙人。大家其实都知道有些话是不能说的，有些话是要拼命说的，但我总是控制不住自己的嘴。他就跟我说，我们CEO俱乐部群里的一个客户，是一个几千万粉丝的财经大号。他也写过非常多的书，也知道中国的舆论红线，管制的红线，也非常知道大众的情绪在哪里。但是有些时候他就是会拼命的探测那个红线，以至于他加入我们CEO俱乐部群开始，已经被封了两三次号了。我们就在探讨为啥。今天我终于知道了，就是因为当 IP 做老板的时候，他就是在向死而生。他就是在向着死亡的阴影前进，在做内容。一、IP 让人喜欢，就是他的性格表达。为什么 IP 做老板，公司可以吸引这么多的粉丝，可以做成规模？是因为 IP 他在用自己的生命经历，用自己的能量情绪，用自己的性格吸引了一些关注他、支持他，甚至愿意为他付费的粉丝。他被这些人支持，是因为他的能量和情绪有吸引力，这是硬币的一面，这一面叫做 IP 之所以能成为IP。因为他本身的生命就有值得追随之处，无论是他的能量，还是他的故事，还是他的性格，还是他所代表的人群对应的价值观。但是硬币的另外一面就非常的悲惨，由于他必须要成为他自己，所以他有些时候非常的飞蛾扑火和上头。我们很多时候都会被劝告，作为 IP，我都会被我的操盘手劝告，被我的助理劝告说老板能不能够冷静一点？能不能够理性一点？有些话能不能别说，然后有些话能不能多说？但是我想说的是， IP 之所以能够成功，就是因为他的脑子是被自由的灵魂所控制和绑架的。二、 IP 一旦理性，就是死亡的序章。IP 一旦理性，他就是会必死无疑。因为他杀死了自己最吸引人的一面。这是一个非常朴素的平衡，当 IP 失去了创作和表达的自由。他将开始走向死亡，他将开始 EMO，他将开始失去创作的动力以及创作需要的生命力。我彻底理解了为什么那个财经大号会永远被封，永远重启，永远在正在被封的路上。因为他的价值观和他想要表达的观点，表达的感情的那种洪荒之力就是他之所以能成功的核心原因。所以我就跟我团队探讨，以后别限制我表达什么，可以在事后给我删稿，但你不要在我录视频的时候跟我唧唧歪歪。不要在我写朋友圈的时候唧唧歪歪。不要在我写公众号定排期的时候，定选题的时候给我唧唧歪歪，但这又是弊端。三、做 IP 就别做 CEO 了。因为当老板让一个员工闭嘴的时候，员工就只会静默无声，而且之后就永远不说话了，那就没有人给我风险控制了。这就是 IP 公司一定会走向暴雷的核心原因。IP 既是老板，又是一个创作者，他就会无限纠结，无限拧巴。作为创作者他必须是自由的燃烧，就意味着混乱和无序，而老板就是要做 CEO，就是要管理，他怎么能够兼容呢？所以我对的解决方式就是从此之后我不会再承担 CEO 的角色。让我们公司 CEO 成为一个联席 CEO，如果没必要，那就是我的合伙人们成为业务线的 CEO。我只做创作者，不做 CEO。一定要谨慎使用老板的独裁权，这样希望我们的 IP 能够多活一日。IP 的本质属性，就让 IP 蕴含着巨大的爆雷风险，这是我最近才想到的。各位 IP 们，向死而生，面向暴雷，面向封号，一起且行且珍惜吧！！！！！！！！-END-群响这个小公司，已经 5 个年头了。未来，我们还将持续和客户在一起，和年轻创业者在一起！6.25-6.28 群响五周年庆直播我将邀请「白小T创始人张勇、叮叮懒人菜合伙人林郑焕、京润珍珠董事长周朔、众多垂直领域超级个体户」 20 位这个新新时代的创业者和我一起分享自己跌跌撞撞的创业成长史，放送五周年超级福利，立即扫码预约！
