大厂打工人转型的四个路径，请参考。 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 这是刘思毅的第1680篇原创持续日更，做特懂流量的创业者今年开年很多人问我，我一直在说新个体，不会只有 IP 吧？当然不是，赚钱的新个体如果只有 IP 的话，那不会做 IP 的，不会做台前的人，i 人岂不是完全没有出路了。今天这篇文章和大家认真盘点一下，到底有哪些新个体在赚钱，到底有哪些新个体的方向是可以持续赚钱的。我所说的新个体是以下四类人：一、最典型的就是 IP 本身IP 本身是最容易做超级个体的，因为他可以通过自己的肉身力量。通过内容营销，通过直播间，通过朋友圈和短视频完成所有的交付、消费还有流量获取。从获客到如何卖出去，再到如何交付，都是他本人录课，他本人直播，他本人去获取流量，所以他就是以自己为中心的一个工作室。我就是个非常典型的这样的 IP。超级个体户 IP 自己当老板有一个非常好的好处，就是他是最了解自己的操盘手，所以他自己既是 IP 又是操盘手本身。这样的超级个体户模型，好处在于利润率特别高， IP 会过得非常爽，但坏处就是容易费命。 IP 基本上就是很难持续，我自己持续了 5 年，我都不敢保证我还可以再干 5 年。二、夫妻老婆店型超级个体户老婆和老公一起合作，一般老婆是在台前的那个人，老公做 CEO，或者做操盘手。这样的 IP 和操盘手的结合有非常大的好处和坏处。好处就是夫妻同心，其利断金，你不用担心 IP 和操盘手相互撕逼跑路，因为你跑路也 AA，现在也 AA。所以两个人 1 + 1 大于 2 是必然的，这是操盘手和 IP 合作的最优解。坏处就是容易谈崩。有很多人因为夫妻、老婆店合伙而分手甚至离婚的案例。因为生活在一起并且工作在一起之后，你的生活和工作都没有距离，人很容易崩溃，男生女生都容易崩溃。在这个模型中，夫妻相处在生活上是需要艺术的，生活和工作本身都非常需要距离感，这种处理起来其实还挺麻烦的。三、操盘手超级个体我觉得这是最引人入胜，但是最容易被人忽略的一类超级个体--操盘手超级个体。操盘手必须非常擅长发掘某一个赛道的核心 IP，和他合作给他灌流量，给他做投放，给他激发情绪价值。让这个赛道的所有 IP 和他合作都能多赚好几倍的钱，于是他成为了这个 IP 的一个明星操盘手。一旦有了自己的案例，他就孵化几百个 IP ，只要成功一个就能赚 200 万，三个就是 600 万，所以很牛逼。操盘手超级个体和 IP 超级个体有一个核心的区别， IP 的超级个体需要表达，需要代表一类人。比如我代表的是年轻创业者，年轻新个体跑酷，操盘手其实需要的是冷静、理性的思考、判断、评判。IP 是文科，操盘手是理科。如果要做操盘手，你是非常理性冷酷的，你要接住 IP 的能量、流量，还有 IP 之外的很多擦屁股的事情。但你不能说你很烦，因为 IP 很烦，才有你赚钱的份儿，这是操盘手超级个体。四、专家型 top sales 超级个体我觉得这是很多普通人可以努力成为的超级个体。在很多高客单价领域，比如说保险，比如说理财，比如说移民，比如说金融，比如说房地产，比如说汽车。这种高客单价领域是需要通过专家人设和长期运营这个行业建立信任的，建立信任之后你的客户池会不断的转介绍。于是你就拥有了源源不断的流量。比如说，人的一生有没有几次房产交易，但是他的朋友每个人都有一次，那就够吃了。豪宅就是一个例子，老纪二手豪车也是一个例子。还有就是房产、身份中介、理财、保险都是例子，这个领域非常需要你持之以恒的耐心去科普，耐心的建立信任。一次次 say hello，一次次添加，一次次的成为朋友，不要预期立刻转化，就是日拱一卒，不期速成。先建立信任，然后再做生意，先交朋友再做生意。但核心是一定要服务超级有钱人的，极致就是做某一个有钱人的家族办公室。我有一个姐姐，她做的是刘慈欣的南极旅游的助理。刘慈欣的一切都由她包办，那一辈子就不愁吃、不愁穿了。当你自己可以锁定一个 super reach，对你的信任就够吃一辈子了。巴黎的奢侈品柜姐也是一样的逻辑。所以，超级个体不仅是 IP，理科生、文科生都可以做。今天的超级个体需要的是你找到你自己的方向，你自己的行业，你自己的赛道，以及符合你自己性格的事情。持之以恒，持续性的用服务、用内容，让别人看到你的存在，然后信任你，然后完成成交。超级个体的时代讲求的是持续性，而不是爆发性。无论是 IP 还是操盘手都是如此，一起共勉，一起在新个体时代疯狂搞钱！！！！！！！！！！-END-我是群响创始人刘思毅，有10000＋创业者会员。每天访谈一位创业者，聊偷流量搞利润。有很多不方便公开发公众号的关于搞钱案例、关于流量、关于创业资源我都分享在朋友圈扫码加我本人微信👇围观我每日15条朋友圈
