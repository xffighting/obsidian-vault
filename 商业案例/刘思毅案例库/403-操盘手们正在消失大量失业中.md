操盘手们正在消失，大量失业中。。。 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 这是刘思毅的第1670篇原创持续日更，做特懂流量的创业者昨天和我非常熟悉的操盘手聊天，他跟我说他们现在没有办法以操盘手的身份来开启所谓的创业公司。在 21 年到 22 年，大众刚刚开始抖音入局的时候。每一个但凡在抖音新媒体上拿到过一点点结果的人，都可以用自己的结果去杠杆很多的潜在客户。哪怕你的客单价并不高，就两三万一个月，但你接到了 10 个客户就是二三十万，赚一波钱三个月。被服务的 IP 没有成功他就离你而去，或者客户自己放弃了，或者成功了把你抛弃了。但无论如何你一个月就是二三十万， 3 个月就是 100 万，作为一个操盘手的第一桶金就出来了。我跟这两三个 00 后的成都的抖音操盘手聊。他们说其实他们之前已经操盘过 1, 000 万粉丝的完整的变现流程。而且真的知道怎么拍所谓的视频，真的知道如何能够调试 IP 的状态，真的觉得自己又便宜又好用，但现在没生意了。我受到了非常大的启发，我原先并没有那么确切得出这个结论，我现在可以确切得出一个结论：操盘手已经死亡，操盘手即将大量被淘汰。代运营作为一个商业模式也会被直接淘汰，我详细推演了一下，有以下几点结论，欢迎大家撕逼。特别是操盘手和MCN，还有代运营公司老板，我需要你们的批判。第一、没有订单了。今天是一个 IP 时代，而且越来越多人知道必须要做 IP ，但是大家都觉得 IP 是一个公司核心业务，不想要再放给其他人做了。其他人必然是割韭菜的，即使其他人主观上是要为我好的，但客观上就无法给我带来结果。因为团队本身不一样，老板越来越发现 IP 很重要，而且 IP 必须要自己做的时候，他就不会再放给别人了。所以江湖上的订单少了非常多。原先代运营订单很多，是因为市场上没有成熟的候选人。没有那么多剪辑，也没有那么多的所谓的 IP 内容运营和那么多抖音小红书运营。就这些螺丝钉岗位并没有在所谓的招聘市场流通，取而代之的就是一个集合了拍摄、素材剪辑、选题以及投放运营的集合的团队。他胜在所有的环节都略懂，就叫操盘手了，江湖上这样的操盘手大概有差不多十万个。这拨人在 21 年和 22 年是赚到一些钱的，有少部分人甚至一年赚到过 1, 000 个 w。但现在基本上都被淘汰了，就是因为老板不需要了。而且老板找得到萝卜白菜价的人当他的螺丝钉。第二、大部分做 IP 老板，要么成功了，要么放弃了。老板们一旦尝试过，就知道自己不是一个 IP 的料了。我现在我觉得我自己最好的操盘手就是我自己，当一个 IP 真正成功拿到过所谓的播放量或者所谓的表达结果的时候。你就会发现其实你是你自己最好的操盘手，没有之一。你只需要有人帮你拍摄素材，有人帮你去拿机子的，有人做你听话的助理，有帮你剪辑，有帮你做运营的。这些螺丝钉完全可以按照你的意愿成为你自己的组件。你并不需要所谓的 IP 操盘手帮你 cover 很多东西，因为你就是自己最好的操盘手。成功的老板是这样，那失败的老板觉得自己不能 IP 的，他也不需要这个角色了。所以真正牛逼的老板自己成为了操盘手，让操盘手操无可操。第三、世界上最牛逼的操盘手还是去自己当老板了。这是操盘手的出路，要么自己当老板做一个盘，自己成为 IP。要么自己成为 MCN 老板选择一个类目和赛道，把这个方向上的 IP 签了，或者娶一个 IP 成为他的操盘手，然后利益绑定。它不是代运营逻辑，它是真实商业逻辑。这需要你完成交付、流量和销售，然后把自己的公司做起来，不太可能存在代运营不断的拿到现金的好生意了。坦诚讲，代运营是一个非常好的生意，因为它没有特别大的保结果的风险。他虽然吃💩很累，但是确确实实给了你现金。现在即将是一个操盘手失业的时代，这是一个代运营会消失的时代。未来一年只有两种角色可以活着，一种是成为 IP，一种是成为赛道 MCN 老板，这是非常非常显而易见的。今天所有的行业都在做 IP，但是所有的行业都不再需要代运营。分享给所有还在做代运营或者还想出来做代运营收割一波的宝宝，没机会了。-END-我是群响创始人刘思毅，有10000＋创业者会员。每天访谈一位创业者，聊偷流量搞利润。有很多不方便公开发公众号的关于搞钱案例、关于流量、关于创业资源我都分享在朋友圈扫码加我本人微信👇围观我每日15条朋友圈
