在群里找创业合伙人，疯了吧？？？ 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 刘思毅宝宝总已持续日更 1759 天！陪伴中国年轻创业者一起赚钱！一起成长！点击下方预约⬇️合伙人是命运的馈赠，求也求不来。如果你作为 00 后、 90 后的创业者，你到底应该怎么选择合伙人？这个问题我无数次被同行的创业者问到。合伙人的类型很多，一种是背靠背型，一种是追随型，还有一种是打炮型合伙人。打炮型合伙人就是项目的合伙人，因利益而来，因利益而散。没有长期合作的积淀和基础，极容易撕逼，非常不建议大家在合伙人方式一夜情。今天这篇文章来认真的和大家说一说合伙人都有哪些类型。以及我自己的合伙人是怎么相遇，如何合作的，以及我看到的私董会爸爸们是怎么招募合伙人。一、我合伙人是前小弟，又如何？？？我在刚刚创业的时候，我觉得我非常孤独。整整一年，我基本上像螺旋仪一样不断的写文案，不断的邀请嘉宾，不断的去约嘉宾访谈，不断的去看看怎么做流量，不断的做收入。我基本上是销售交付，还有流量一把抓，我就觉得我好累。这个时候，我就会觉得，我的合伙人 Toby，Allen 在做什么？为什么都不能帮我分担一下！我没有这么说，但是每次我的情绪是这样的，我在那个时候一直在追求背靠背合伙人，但事实上他们不是。当时合伙人是我的小弟，不是我的背靠背，需要我输出。合伙人的类型有非常多，核心是和创始人的性格秉性有关。我是一个非常自我的人，控制欲极强，占有欲极强，非常自恋，公司有我一山不容二虎。我只能是追随型合伙人，他不能是背靠背合伙人。二、背靠背的合伙人，需要经验非常丰富。你职场人生中遇到的人就很多，你可筛选的就很多，你看得上的人和你走过十年，你们在一起合伙。你们在职场中，在人生中，在家庭生活中，在个人深度的谈话中，你们非常同频，共同进步，共同的利益，共同的经历，共同的认知，共同的讨论，彼此的见证。他就成为了一个旗鼓相当的合伙人，就叫做背靠背合伙人。这时候你只需要 focus 聚焦于你要干的事儿，然后很多时候是布道师，还有一些关键业务。其他你可以把后背交给他们，因为你了解他们，你信任他们，他们也信任你，他们也了解你，而且你们彼此了解。但我就是我们公司最老的，而且我不喜欢和比我老 5 岁以上的人成为合伙人，因为我很容易和比我年轻 5 岁的人同事共事。我想和年轻人在一起，我如果请教别人的话，他们适合成为我的客户，但不适合成为和我的同事。我希望朝夕相处的人更年轻，而且我也不足够的老，所以我需要的是追随型，或者从追随型到背靠背的路程型的。背靠背的好处就是放心把后背交给他们，你没有那么累。但是弊端非常明显，容易撕逼，因为合伙人太强，以至于如果你搞不定他。或者你的能力没有进化，或者你做的这个盘子非常容易出问题有经济纠纷，基本上都是大雷。一个好的地方就是战斗力极强，坏的地方就是战斗力强，因此控制力弱。三、追随型合伙人，绝对指哪打哪。追随型合伙人，首先信任、忠诚以及绝对的为你服务，绝对的指哪打哪，最适合的业务类型就是 IP 工作室，没有之一。 IP 工作室以 IP 为中心， IP 负责内容，带来信任，带来流量。一个合伙人负责销售，负责 GMV，一个合伙人负责交付产品，和 IP 一起共同担当一些内容的角色。这种三角形其实就是刘思毅做 IP 中心， Toby、Allen 做追随合伙人。这个利非常简单，就是长期信任、长期忠诚、没有内耗。弊端也非常明显，就是创始人的上限绝对决定企业上限。这是一个超级个体时代最悲惨的局面，当一个创始人决定退休那一刻，他的企业就是死亡。企业没有所谓的品牌，没有所谓的「IP 停了企业还可以继续」。超级个体时代的企业有一个最大的隐患就是它是百分百创始人发能的，当创始人停止燃烧，企业瞬间熄灭。四、合伙人不是主观选择的，而是命运的安排。以我为例，我现在合伙人 Toby，他是我第一个实习生，他一直跟着我。从实习跟到全职，跟了我两个公司，再跟到我现在的公司。合伙人Allen，也是我当年的一个面包求职的应届毕业生，一直有联系，现在成为合伙人。Allen 擅长销售，擅长经营，Toby 擅长内容，擅长和别人打交道，用内容来输出去，是我的一个小 IP 和副咖。这些人都是非常非常巧妙地衔接在我身上的，而这些衔接其实都是命运的安排。不是我找到了他们，而是命运帮我安排了他们。很多人会问我说，合伙人能不能帮我安排一个合伙人？能不能帮我介绍一个？答案是不能。你需要从欣赏的人群中找，而不是在三教九流中，也不是接触不到一年的人就可以成为合伙人，非常轻浮。合伙人就是一个项目的合伙人，要心态非常匹配，性格非常互补，以及肉体和精神上都非常信任。百分百信任才叫合伙人。合伙人是练出来的，是走出来的。合伙人不是找到的，是命运给你的礼物是选择出来的。-END-群响这个小公司，已经
