刘思毅全球跑酷，有 30 个好看助理在线支持哈哈哈！ 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 刘思毅宝宝总已持续日更 1711天！陪伴中国年轻创业者一起赚钱！一起成长！做 IP，没有助理等于白做，有助理省寿命 10 年，多赚 1000 万哈哈哈哈哈哈。刘思毅宝宝总经历了 5 年时间，形成了一套完整的方法论。什么叫 IP 的助理，怎么搭建 IP 助理，怎么完成助理的新陈代谢以及培训，三个部分。今天全面跟老板们分享一下，如果你也想做 IP ，欢迎抄袭，欢迎切磋。一、到底什么是 IP 的助理？IP 助理不是帮助你完成吃喝拉撒睡的。很多人觉得助理的工作特别低级，其实完全错误。 IP 本身是一个内容创作者，而助理是为了让 IP 能够把核心的时间都用于最关键的生产内容。其他的时间，其他的流程，其他的工作全部由助理来完成。简而言之就是把除了内容创作这个最核心的工作内容之外的一切要素全部分工出去。包括他的选题收集，剪辑，运营，排版以及情绪价值抚慰。助理这个岗位非常关键， IP 的时间和生命能力都是非常有限的时间。一天睡 12 个小时，摸鱼一半时间，每天有效工作时间 4 个小时不得了。所以每一个生命能量用到最关键的那个点位非常重要，用的关键赚的越多。二、优秀的助理体系是什么样的？我自己的助理体系总结一下有以下几类。第一个，生活助理。我有两个生活助理，一个是公司对内的，负责我的人力，财务，公司有关的一些生活费用，比如订酒店、旅游机票、餐饮出差报销。另一个生活助理最重要的任务就是要非常认真负责且懂我。除了钱之外，生活一定是非常需要你鼓励、肯定、陪伴、认可而且安慰的。第二个，内容助理。我的内容助理非常多，包括我的选题助理，不是他帮我策划选题。而是我看到很多合适的选题扔给他，他帮我记录起来，每天都盯着我拍。我自己看到合适的发给他，他看到合适的发给我，我们完成一个切磋，这是选题助理。公众号排版助理不仅是要排版，最重要的是基于公众号的完读率、分享率。好的标题来和我确认每周的排期以及每周的阅读量，每周给私域流量的情况。短视频的运营助理核心不是剪辑，而是怎么用视频号让所有的私域转发，怎么在抖音投抖加引发爆款，负责我短视频的运营逻辑。第三个，剪辑助理。剪辑助理这是非常重要，短视频操盘手最重要的能力就是能不能够足够的了解 IP，在他拍完后快速坐冷板凳，快速输出。我们的剪辑助理有三个，还有两个外包运营服务商，这些东西都是我的剪辑助理。比如我周年庆当天要持续发二三十条，我在大会有一个内容助理帮我拍，拍完之后马上直接百度云盘。剪辑助理的 leader 直接分配任务，直接剪，直接上传，直接转发，这是从内容助理到剪辑助理的工作流程。三、怎么让助理实现新陈代谢？怎么保证助理的稳定？怎么让助理死心塌地跟你？首先要坦诚讲，助理这个工作本身是一个过渡工作。他上可以成为项目的操盘手下，下可以拿着 IP 助理做成一个简历，就是某某 IP 的操盘手，也可以出去创业。在我们公司是一个非常有弹性，有上限也有下限的一个工种，对于我们公司来说，不需要告诉他们助理有多么多么有前途。最重要的是助理本身要承受你巨大的负面情绪，巨大的能量，巨大的正能量，巨大的负能量。负能量就是 IP 有些时候不想弄了，或者 IP 本身有些时候觉得你们怎么都不懂我，你骂他是不是都是傻逼？这种愤怒，这种谩骂其实是没有必要的，但是 IP 不得不找到一个情绪出口，那助理不挨骂谁挨骂。助理每天都很委屈， IP 骂了之后一定要补回来。发发小红包，发发定向安慰，以及经常请他们吃饭，以及经常要告诉他们，你看到他们了，你知道他们很重要了。以及全团队都需要知道助理对老板有多么重要。整个以 IP 为核心的公司，助理本身就会占 30% 以上的工作内容以及团队规模。整个助理团队就是你自己内容营销的中心，实际上他不应该是助理，就是内容操盘团队。只不过他像明星工作室一样围绕着 IP 继续工作，所以叫助理更顺一些。内容助理、剪辑助理、选题助理、公众号助理、生活助理整个助理形成了刘思毅宝宝总每天分享 15 条朋友圈。每天日更两篇公众号每天拿到 20 万视频号播放量，以及每天稳定进量 200 个创业粉丝的一个基础。这就是我们自己今天这个 IP 时代必备的助理体系，如果你也想做 IP，欢迎抄袭刘思毅的 IP 体系，一起赚钱，加油！-END-我是群响创始人刘思毅，有10000＋创业者会员。每天访谈一位创业者，聊偷流量搞利润。有很多不方便公开发公众号的关于搞钱案例、关于流量、关于创业资源我都分享在朋友圈扫码加我本人微信👇围观我每日15条朋友圈
