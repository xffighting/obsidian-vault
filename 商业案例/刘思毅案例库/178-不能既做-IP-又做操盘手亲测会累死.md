不能既做 IP 又做操盘手！亲测会累死！ 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 这是刘思毅的第1323篇原创持续日更，做特懂流量的创业者回首创业之路已经五年了，在这五年间看了很多 CEO 做公司的思路和模式，我发现其实大部分的 CEO 都有非常大的痛苦，他很渴望自己既是 IP 又是操盘手。我们这个行业，从我 2019 年创业到现在为止，其实大部分的公司都是一个模式，抖音 IP 获客 + 私域精细化运营 + 行业。解释一下，就是一个 IP 跑在前面，全方面全领域的在各个渠道获得流量，流量在私域被精细化标签，精细化收割获得利润，但这一切的一切，建立在一个天时地利人和能够赚到钱的行业上。但是，这个模式有一个天然的弊端，就是作为这个公司的最高意志，你到底应该在那里发力？如果你自己既当操盘手又当 IP ，或者拥有一个 IP，你自己做操盘手CEO，或者你就是IP，然后兼职操盘手的角色，把公司做大做强。100 万也好， 500 万也好， 5000 万也好，都可以。我看到的最大的超级个体户，以这种模式成型的，也就是 5000 万的体量，虽然看起来很赚钱，但是这类公司永远是非常痛苦的操盘手公司。一、老板很难绑定 IP ，IP 成熟了必然跑路。如果你选择做操盘手CEO，你最大的痛苦是发现 IP 永远想跑路。IP自己一旦被孵化出来之后，无论你多让步，让步多少个百分比，还是多么努力的跪下来， IP 一旦成为一个超级大主播，他就一定想跑路，他一定想做自己。圈内有一个非常知名的网红，他原先其实是一个名不见经传的小孩，然后被自己的投资人孵化，投资人很看好这个业务，于是和他成为了合伙人，既是投资人又是合伙人，投入自己的时间和金钱，所以说这个人占大股，而这个小孩占30%，合伙人和投资人占70%。这个小孩一朝成名，拿到了 500 万的粉丝，成为了小股东，所以顺理成章的成为了这个公司的顶梁柱，原先投资人投入的几百万其实在这个小孩面前都是他的束缚，都是他的凭什么和意难平。这个小孩就不想管公司的业务，他沉迷于自己做内容，夯实自己的羽翼，提高自己的影响力，一心一意只想有朝一日自己有足够多的筹码跑路。于是他真的跑路了。二、IP 一旦跑路，必然和公司两败俱伤。跑路之后，投资人、合伙人作为一个操盘手的角色，和这个 IP 的角色两败俱伤。因为投资人和合伙人拿着这个小孩的抖音账号，它可以直接改变这个小孩儿所有的资产的性质，蓝 v 化、改名，以及曝光这个小孩儿所有见不得人的丑闻，所以 IP 就脏了，脏了就很难起势。对公司来说必然是巨大的损失，因为大家关注账号都是冲着 IP 来的，没有了 IP ，流量断崖式暴跌，没有内容就没有观众，所以用户纷纷离开。这个小孩儿，他拥有自己的脸，IP是一个可以动的生命。所以说这个小孩儿自己出去跑路，就可以直接号召一帮自己的铁粉，又做一个自己的号。但是起号何其辛苦，要熬过最难的冷启动期，还要被原来公司不断的泼脏水，不断的打压，不断的经受身体和精神的压力，然后要自己对产品，自己去对商务，开始做原来公司操盘手要解决的一切杂活。这就是一个两败俱伤的结局。IP 的操盘手公司做操盘手的是 CEO 的这种情况，最疼的就是 CEO 被IP背叛跑路了，而且这个 IP 竟然是这个公司 95% 的营收，怎么办？没有办法，IP必然跑路，且 CEO 也没有办法阻止他。三、老板做 IP 的终局是发现公司永远做不大那另外一面像我这样的情况，我是 IP，我即公司，这种模式本身最大的痛苦就在于公司永远长不大。长不大分为两个方面：第一个是因为一个人本身的精力时间有限，如果说刘思毅，百分百的刘思毅做直播，百分百的刘思毅做短视频，百分百的刘思毅写朋友圈，我如果这样做公司，我公司可以翻 3 到 4 倍，但是刘思毅只有一个人，公司就会非常受限于我的时间和精力。这是一个做不大的原因。还有一个做不大的原因，大家都不敢说，是作为 IP 本身的自恋。一个 IP 为什么可以成为 IP 是因为他想表达，而表达本身是需要自恋和存在感诉求的。表达就是希望别人听到你、别人认可你，当你自己是这样一个人的时候，你其实是很难给予你的团队成长空间，你的团队会感到非常窒息，你的团队会非常非常觉得你是个傻逼，你自己却不自知这一点，直到你大部分的关键人物、你的糟糠之妻、你的合伙人全部离你而去，你成为了一个孤家寡人，你那个时候才认识到你真的是个傻逼。所以说无论是操盘手 CEO 还是 IP CEO，好像都有各自的毛病，操盘手 CEO 害怕被背叛，IP CEO 一定会众叛亲离。每个人都应该选择一个平衡节点，但一旦平衡，好像就不能有一个非常明确的业绩突出了。你一旦操盘手想做IP，你就做不好操盘手IP，想做操盘手就做不好IP，所以说你只能接纳。人生就是这么遗憾和残缺的。如果说你们是操盘手，你们自己怎么克服 IP
