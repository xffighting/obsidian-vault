刘思毅真的要穿西装拍视频？有必要吗？？？ 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 这是刘思毅的第1390篇原创持续日更，做特懂流量的创业者网红校长，我的表哥是知识付费鼻祖，代运营鼻祖，各品类变现直播鼻祖，大厂浪潮式发售鼻祖，到今天没有任何人能够超越他抖音的知识付费零投放，单日卖了 424 万的神话，真的非常非常非常牛逼，经过半年的这个躺平，他在 520 的时候在大理做了一个中国知识 IP 不焦虑大会，他说他自己的大会他都没有分享过内容，这次我们的聊天是他最近半年关于 IP ，关于社会的思考，这篇文章记录我们聊天的精华内容，欢迎阅读转发！！一、做爆款的核心：必须站边！网红校长：为什么我能做出那么多爆款？最狂的时候每天抖音上有 3000 万人听到我的声音，有 3000 万人看到我办公室的爱马仕乘的椅子。原因就是当你要做 IP 的时候，最核心的东西叫 take side。你在所有做 IP 的过程当中，只要是观点输入类的，你一定要有假想敌。你需要很明确的知道这一帮人是我要干的，这一帮人是我要留下来的。制造焦虑这件事情能成立，就是我站在了所有身心灵的那一边了，其实你需要解决大家焦虑，不是制造焦虑。制造焦虑是一定会反噬的，很多你讨厌的人他现在还站在那，只是时间还没有到。给大家讲一个案例，我的一个会员，杨紫说早教，直播也不投放，一天能卖 30- 50 万，这是绝对的天花板中的天花板了。我们公司也有一个早教的 IP，他的直播在线就是 9 天做到 1 万人在线，但变现只是杨紫的 1/ 10，就开始非常焦虑，为什么同样的直播在线，然后为什么别人能卖十倍，他只能卖2万、3万？有一天特别难过，我和他说，杨紫在公众号时代他就已经帮助了 100 万家庭了，但是他那个时候是做什么呢？他那个时候是帮助这些平台写课的，所以他拿到的分佣非常非常少。 可能录一节课就 2000 块钱，但是他在公众号时代已经帮助了超过百万的家庭了，而那些钱他的课已经卖了过亿人民币了，但他可能自己就拿到 200 万，她可能就拿到个录课的费用，那钱被平台赚了，所以当他这么强的人来到抖音的时候，他已经有自己的福报了，就是她踩中红利，但是跟她没有关系，我就跟我们的这个博主说，你想象一下每天有 10 万家庭因为听了你的直播而养成了科学的育儿观。今天不报给你，明天也会报给你，不报到你身上就报到你孩子身上。这个经营价值观和抖音操盘手江湖流传的是要应割尽割快割的逻辑是相反的，恰好是相反的。二、环境很重要，要回归自然激发 IP 的生命力。网红校长：我最近又回到抖音的时候，我发现这些商业 IP 一天播十几个小时，我都感觉他妈张一鸣快黄了。你看大蓝，我真是觉得太牛逼了，他每天早上 6 点多开始播，一直到凌晨两三点他还在那播。 这几天的困惑就是所有的人都拼了命在播，我也不知道为啥。大家都觉得好像要应割尽割，应收尽收，未来不会特别好。但是我不这样认为，我觉得教育是可以一直做下去的事情。商业 IP 确实是在长期的疯了一样直播。所以我只能走另外一条路，我们甚至准备做线下的学校，当然我做的那个可能它会有吃喝玩乐都可以，就叫 IP park。它会有很好的环境，你能在在森林里面，你能看到洱海。我们甚至开始做线下的这种孵化。然后这是第一件事情，可能是明年开始启动这个事情。但是今年下半年要做的事情是我带着这些 IP 去喜马拉雅山上拍视频，然后去可能去草原，去沙漠。当你坐在屋里说的话跟你在看到大山大河说的话是不一样的。所以我们可能先徒步走他个 20 公里，完事以后在雪山下面。哇，你说出来的话能一样吗？为什么我要去雪山，为什么我要去草原，去沙漠拍，因为本质上我们最终要跟地球链接。当你链接的越深，你才能够真正的知道自己要什么，你才能看见自己。你说出来的话才能真正地打动别人。三、想做成功的 IP 掌握这几项就行了我告诉你， IP 要成功，其实就这么几项：第一是形象，当我们刷到短视频的时候，第一看到的都是形象，形象不是长相，是这个人整个包装，包括这个后面的背景，他像不像他想要表达的那个人。第二是声音，声音也是有能量的。刘思毅的声音要往下沉一点，你只要把这一点提升了之后群响的业绩翻 5
