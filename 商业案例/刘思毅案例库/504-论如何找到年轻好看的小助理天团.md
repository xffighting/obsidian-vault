论如何找到，年轻好看的小助理天团。 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 刘思毅宝宝总已持续日更 1751 天！陪伴中国年轻创业者一起赚钱！一起成长！点击下方预约⬇️谁都想要年轻好看有能量的助理，创业者都需要一群好助理，怎么找？做 IP 的旅程，就是一生的内容创作，我助理们把我的朋友圈以及我喜欢的人的朋友圈，每周都给我弄成一个小册子。让我放进包包里，随时想录视频缺乏素材的时候，就翻一翻。全世界最好的 IP 助理团队，原来是我在背后，我感动哭了，走过 5 年，终于开出花，怎么培养的助理呢？这是新个体创业第 13 弹的内容。第一，什么是助理？为什么要助理？第三，怎么寻找一个好的助理？第三，怎么沟通、怎么培养助理？一、什么是助理。简而言之就是在文字内容上的小编角色，在短视频时代，变成了助理，不只是生活助理。我这里讲的助理同理于内容编辑团队，只不过在这个个人主义、IP 驱动的时代，IP 经纪人工作室的老板就是 IP，经纪人就是 IP 自己。助理组长可以叫经纪人，但本身其实叫助理更合适。解决一切除了内容创作核心生产流程之外的上下游关系，以及选题、剪辑、运营等事项的负责人，都是助理的角色。基于这个助理的定义，我有，直播助理、公众号排版排期助理、视频号运营助理、抖音运营助理、选题助理。虽然我们鼓吹操盘手，但在我们公司，我压制 IP 操盘手这个概念，无限自我暗示，降低对操盘手的预期。反而得到了许多惊喜，这些图里的手册，都是助理团队自己探索出来的。为什么我们需要助理。因为内容创作和商业变现会耗尽你每一分每一秒的心血，所以一切没必要自己做的事儿，都应该甩给你自己的团队。IP 助理团队，自己管、自己带，全面服务 IP 的内容从创意产生到运营。二、什么才是一个好的助理？第一，年轻。00 后真的是底线的底线，因为这一代新个体并不老，而且给出的工资都很惨淡。助理是起点，不是终点。第二，互相有信任感。信任感来自于相处，你能不能把员工真的当兄弟呢？这挺难的。第三，有情绪价值的输出的特殊面。有些是因为好看，有些是因为靠谱，有些是因为默契，有些是因为妥帖，有些是因为从小带到大。都不一样，我觉得好助理还是要跟的久。第四，心力强，能适应多变的想法和反馈。IP 都是疯子，不能适应他的过山车心态，不能成为一个好助理。第五，特别警惕很懂的人。不需要招聘很懂的人，我觉得助理就不应该是操盘手的角色，应该更是一张积极的白纸。第六，看感觉，每一个人的感觉真的非常不一样。IP 本人得习惯，否则很难，喜欢这事儿，妙就妙在只有看了才知道。三、怎么培养？怎么沟通？1，要探索出自己的内容业务流，在这个业务流程中，安排好助理的标准动作。比如说我的业务流是朋友圈到短视频，再到社群、公众号，然后再是抖音视频号和小红书。每一个助理干什么基础的流程，要自己定义，没有一个通用的助理工作，助理就是跟随 IP 的安排的。2，指令不明确，是 IP 的责任，助理不是万能的，你要让他做什么，请说清楚。很多 IP 会有一些异想天开，然后让助理去随意发挥，这完全不行，因为他们也不知道你要什么。人类的悲欢从不相同，你的情感得不到他们的响应，要是需要他们帮忙，要把他们怎么做说清楚。因为内容创作太非标了。3，发脾气要道歉。IP 真的太容易发脾气了，因为太多内容创作的情绪积压，没有情绪就做不了创作者了。但是有脾气发了之后要承认，抱歉或者一杯奶茶或者情绪价值补回来。助理应该是承受 IP 情绪最多的人，要对他们在情绪上，补偿多点儿。4，不要期待助理在核心创作中贡献核心价值，做不到，除了你谁都做不到。助理的题中之义就是帮助帮助帮助帮助，你是贡献核心价值，助理是提供一种切磋的物料。任何让助理写文案的创作者，走不远的。5，安排好新陈代谢和晋级通道。一般助理在身边不会超过 3 年，一个年轻的助理要向前走做更多事儿，或者成为你的经纪人的头，或者做其他项目，或者就去其他公司。ta 要向前走的话，需要你给 ta 明确的托举，当然你的托举也期待 ta 用他的价值来回应，但总是你先走一步。6，多听他们说他们的感受，因为他们是最了解你的受众，也是围观受众最多的人。助理每天都在剪辑你的片子，都在编辑你的票圈和文章，他们会不由自主地思考和感受。让他们批判你，让他们说直觉，让他们说从数据中、从评论区看到了什么。7，不要让助理团队成为整个公司的孤岛和太子党。让助理和团队们融合，让助理们也参与主要业务线的支持，让助理也参与常规月报、周报和业务会的汇报，让助理也是同事。8，不要把救命稻草放在一个助理上，让助理新陈代谢。无论是 IP 还是 CEO，这个时代的新个体一但要自己做老板 + 内容创作者，都会不断和助理合作。不要因为喜欢就觉得不可替代，这个世界上没有不可替代的事儿。这个时代啊，是好的，大厂没有那么多机会，会有江湖上很多没有被发现的珍珠，用心找到、用心培养和沟通，可以贡献巨大价值。以上。-END-群响这个小公司，已经 5 个年头了。未来，我们还将持续和客户在一起，和年轻创业者在一起！6.25-6.28
