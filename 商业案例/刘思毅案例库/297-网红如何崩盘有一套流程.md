网红如何崩盘，有一套流程。 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 这是刘思毅的第1558篇原创持续日更，做特懂流量的创业者网红如何崩盘，有一套流程。一、在西贡的时候，见了一个被封杀的中国网红。这逼在图文自媒体时代就开始红，但偶然情况，抖音口出狂言，被乱拳捶死，只能退居幕后的古早网红。她跟我说了一下中国网红会如何崩盘，让我汲取教训，向死而生，面向封号而活。非常醍醐灌顶的思考，记录下来，分享给所有做 IP 的宝贝。二、首先，要明确自己怎么看 IP 业务，别把自己也糊弄过去了。IP 最值钱的是什么，是影响力，越做越有，但是在任何一个国家，影响力的最大垄断者是谁，是国家。谁影响国家的基本盘，谁在和国家抢生意，就是在找死。所以 IP 就是一个生意，不要自恋觉得这个生意需要任何为爱走天涯。或者飞蛾扑火，为自己的价值观浴血奋战的道德高尚感。出来卖的，就别觉得自己不是婊子，无论是哪个品类，都是婊子。她说你做的特别好，客户就是爸爸，摆正自己的心态，国家是祖宗，平台是祖宗，消费者也是祖宗。影响力是这三者给的，而不是你自己赚的。这个心态放低，被封号也不会有特别大的刑事责任，否则很容易进去才缝纫机。须知，做 IP 是为了不做 IP，不终有一日不能做 IP，要么你被人民群众和官方封杀，要么你就是自己赚够钱了，或者心力消耗殆尽，然后放弃。三、她曾经和国内数一数二 MCN 的签约，做过颜值网红，之后成为情感博主。做直播的时候，和女人和男人都可以唠嗑，所以她的大哥很多的，她说也别对任何人动真感情。网红没有任何感情，目标就是照顾好所有的大哥。而且网红本身的消耗是很大的，因为你要持续重复，而且是不断用服务的心态和生命力来换钱。要么堕落到没有力气自我反省成长，要么就是精气神消耗到尽头。不是流量问题，是心智问题。她说她自己被封号，就是因为直播了几百次，真的烦，就是突然上头，说了一句实话。说，穷人咱就别想 XXX，然后眼神特别嚣张真实。直播间在线 1000 人而已。但是不幸的是被录屏了。四、网红崩盘三步走。第一步，被录下来物料，短视频时代的物料就是直播和短视频，喷子上传，发酵成热搜。第二步，民间公众号、吃瓜 IP、都市报媒体，不断发酵跟进，讨论，不断有立场批判。喷子涌入你的直播间和你的短视频账号，打乱你的所有运营节奏。第三步，官媒下场定调，尚方宝剑下来，平台封号，抖音封号，其他平台一律跟进。跟随甲方，此 IP Over。都是遵循这三步，没有什么翻盘的方法，除非你和抖音关系够好。抖音可以隐藏你的热搜关联账号，让你的主要账号不要被搜索到，马甲号废弃。官方媒体的评论与否，是一个玄学，也需要非常强的及时沟通和强悍渠道。基本上民间 IP 一旦上热点，板上钉钉的违背「社会主义价值观」的热点物料，必死无疑。五、基于此，她跟我说了几点 IP 心法 —— 1，第一时间回应所有黑料，用户拥有记忆，因为你不回应，就会被动陷入算法推荐的漩涡。2，如果有可能，钱才是最终结果，要那么多粉丝和影响力，不如要钱，要想明白这一点，影响力是一种双刃剑。3，向死而生，随时准备被封号被举报被拉出来鞭尸。反正 IP 这个生意，在今天的算法推荐时代的中国，就是别着脑袋做生意。以上，老子决定再也不参与任何一场抖音的争斗和创作。远离算法推荐的抖音，作为创作者也好，作为用户也好。太他妈喧嚣了。这个姐姐在西贡隐居了，赚够钱，生孩子，做幕后，出海了哈哈哈哈。她眼神里的沮丧，用了整整 1 年才走出来，她说情绪伤害极大，因为所有差评、人肉、威胁，都蜂拥而至。一个人很难接住，但这就是 IP 的负面，不是吗？写完我心惊胆战，我不敢创业了姊妹儿们。「摇人时刻」1 月，2 月，3 月，4月，刘老板全球漫游月，行程涉及：巴厘岛、悉尼、墨尔本、奥克兰、伊斯坦布尔、巴黎、柏林、罗马、里斯本、阿姆斯特丹、哥本哈根、法兰克福、瑞士、维也纳、纽约、旧金山、洛杉矶、亚特兰大、华盛顿。请以上这些地方的华人创业宝宝、网红宝宝、学生宝宝和我链接。我可以给你分享中国在发生什么，你给我分享你在这个城市在发生什么。加我微信。👇
