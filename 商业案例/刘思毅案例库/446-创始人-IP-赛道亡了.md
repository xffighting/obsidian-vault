创始人 IP 赛道，亡了。。。 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 刘思毅宝宝总已持续日更 1720 天！陪伴中国年轻创业者一起赚钱！一起成长！兄弟姊妹儿们，创始人 IP 这个赛道算是没了。周鸿祎、雷军还有一系列的汽车创始人的入局，我们可以正式宣布创始人 IP 这个赛道正式灭亡。未来不存在所谓的创始人IP，只存在细分垂直领域的一个有短暂生命周期的 IP。周鸿祎、李斌、李想、雷军做创始人 IP 之后，我们的创始人 IP 真的可能就没有机会做了。原先我们做的所有选题，在他们面前真的就不值一提。他们是真的经历了 10 亿美金级别以上的战斗的，自己的业务也非常茁壮。真正的创业者开始下场做创始人 IP 的时候，我们这样的小卡拉咪讲一切都没人听，都觉得是一个演小品的脱口秀演员。原先我们做创始人 IP 仗着就是真正牛逼的创始人，从来不下场。但现在他们也认识到这个时代，抖音也好，视频号也好，小红书也好，都是创始人必备的媒介工具。我们怎么办？作为一个小老板，我的策略是以下几个，跟大家分享一下：第一，不再自诩为一个老板。不再说自己是一个创始人 IP ，而是非常谦卑的自称一个小卡拉米创业者。一个正在被员工暴打，被形式暴打，被流量暴打，被大众舆论暴打，被所有人暴打的一个创业者。分享自己的暴打经历以及的血泪史，更谦卑。当然你可以讲创业的内容，但是你不是一个指教的姿态，是切磋，是平行的记录，是一种分享的视角，千万不要以高打低。今天这个时代，雷军、李想、李斌、周鸿祎这样的创业者一旦是指教，别人都会骂死他们。大众舆论才是真正的审判者，所有的弱者集合起来，所有的沉默大多数它集合起来，在算法投票机面前，构成了对所有强者的终极审判。强者装逼必被网暴。所以我们这样根本不是强者的人，何必装那个逼呢？第二，不要贩卖悲惨的故事了。反复讲悲惨的故事，只会让你变成一个非常可恶的无聊的人。可恶在你在反复用悲惨的物料骗流量，无聊在你的故事没有人有耐心听第二次、第三次。你的人设不是通过故事和经历反复陈述，而是通过你不断变化的生活内容来展现。我在美国旅居，我就分享我看到的美国华人创业者以及中国华人创业者。美国好的生意模式和中国好生意模式的差异故事，以及美国人和中国人不同的思维模式以及沟通的方式。以及美国的教育以及中国的教育的差异，以及美国人的消费心理，以及美国的零售生意，线下生意，还有美国的互联网，美国的AI。我看到的一切都是我的物料，这不是我的故事，而是与大家相关的，可以对大家构成帮助的一些物料生活案例。还有我的成长，我在创业裁员，我在创业分钱，我在直播失败，我在视频号直播成功，那我就分享我成功和失败的故事案例和复盘。真实的故事，真实的案例，真实的复盘。让大家都能够从你的分享中得到一些交付，一些价值，一些启发、一些共鸣，是共鸣，而不是教育。注意，眼光从来不是自上而下，而是平视，而是分享，而是价值。第三， IP 这个赛道真的亡了。但有一个品类的 IP 非常非常持续，就是细分垂直，越做越细分，越做越垂直。基于细分垂直赛道分享你的专业内容，并且基于你的成长持续更新你的专业内容。比如以刘思毅宝宝总为例，我的细分垂直是我陪伴的是 90 后、 95 后、 00 后的这帮年轻的。在中国的抖音、微信、私域、互联网吃刨钱拿到中拿到第一桶金，第一个 100 万人民币的这些年轻创业者。我想陪伴他们一起成长，一起赚钱。我关注的、我经历的以及我讨论的是他们关注的话题，这是我第一个细分垂直。第二个细分垂直是我本身作为一个内容创作者，作为一个私域短视频，还有图文消息朋友圈的创作者。我在创作方面遇到的各种门槛难题，以及我对创作的感受。我如何直播，我如何写朋友圈，我如何写文章，我对流量的拿捏以及对流量的思考。第三个细分垂直是我作为一个 30 人的小公司、小个体户，我在分钱上，我在管理上。我在创业者的合伙人关系上、在销售上、在流量上、在产品上经历的无数个坑。以及无数个我非常非常想要分享给大家，复盘给大家的瞬间，还有案例。这些都是我的细分垂直，不再讲吹嘘的案例和故事，不再秀截图，不再激发焦虑，不再反复悲惨故事树立无所谓的人设。父母不讲，小镇青年不讲，北大不讲，只讲持续不断的人生历程。这是今天周鸿祎这些前辈来做创始人 IP 之后，我们的路径。我们遇到的竞争越来越强烈，但是这仍然是一个最好的时代，因为用户只看人格化的内容。用户只信一个真正在路上，在舞台上表达真正内心的一个人。2024 年所有的老板都必须要不躺平，被的员工和的野心扶起来，否则你必死无疑。你要用 IP 人格化的面向客户沟通对话，这是我们今天流量营销的必修课。和所有正在做 IP 的老板和创业者共勉，表达真正的内心，陪伴用户真实成长，也陪伴真实成长，加油！-END-我是群响创始人刘思毅，有10000＋创业者会员。每天访谈一位创业者，聊偷流量搞利润。有很多不方便公开发公众号的关于搞钱案例、关于流量、关于创业资源我都分享在朋友圈扫码加我本人微信👇围观我每日15条朋友圈
