网红活该被锤，因为太赚钱了。。。 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 这是刘思毅的第1585篇原创持续日更，做特懂流量的创业者做 IP 是为了不做 IP，做 IP 是不可能长久的。前段时间在巴厘岛泳池边和一个退休多年的古早网红聊了一个非常深刻的天，给我做 IP 有一个非常大的启发。她跟我说：刘思毅，做 IP 做网红第一要务是如果你想持续赚钱，就要搞清楚这个生意不能让你失去赚钱的机会。除非你自己知道如何能够防火、防盗、防封禁。啥意思呢，就是我自己心里要有逼数，这个生意只是我存续原始资本的第一步。一、做 IP 是玩火，时刻可能引火烧身。这个小姐姐是原先在公众号时代就非常火的一个人，由于自己一招不慎玩火。在抖音时代被录屏了一个非常犀利的，影响大众情绪的一句话，然后 IP 就烂了。她当天直播被录屏，被放到了喷子的账号，喷子的账号就瞬间火了，播放量一个亿。然后就上热搜，不断的发酵，不断的被很多看热闹不嫌事大的都市报媒体、公众号媒体、吃瓜媒体以及很多抖音的同行 IP 去拆解。于是抖音就下场了，她被平台封禁，官媒就给她定性，就被封号了。二、做网红被人捧着的时候，真的会飘。一个网红自己原先拥有得天独厚的优势，无论是颜值还是内容表达能力都很好的时候。她会觉得自己什么都会，她就真的飘了，她以为她是粉丝的救世主。所以她在直播间才能说出「年收入不足 10 万的人不要成为我的客户」这种挑战大众认知的话。她当时就被直播间的差评覆盖了，于是她心态就崩了。一年之后，她刚刚恢复过来，她跟我说做 IP，一旦你的影响力足够大的时候你就会飘，飘了之后就自有天收。就无论是你自己出 bug，还是被举报，被你的喷子抓住，你总会出现问题。一定要知道自己一定会飘的，飘一定会受惩罚的，这个世界就是这样一个因果报应轮回不爽的。三、时无英雄，使竖子成名。今天中国抖音时代的 IP 就是时无英雄，使竖子成名。最近出了很多事，刘媛媛、罗永浩、小杨哥、辛巴、董宇辉。这些所有的热搜其实都是一个事儿，都是关于大 IP 的，整个 IP 被舆论炒脏了。去年开始，主要舆论节奏其实还是说 IP 其实蛮正能量的，要研究一下怎么能让他帮我们带货，怎么能让他帮我们赚钱。这些 IP 代表了一类人，比如说刘媛媛代表的是小镇青年；小杨哥代表的可能是大山里走出来的普通人；罗永浩代表的是甄嬛传，还了 6 个亿，很牛逼；董宇辉代表的是丈母娘都很喜欢的倒插门。但现在的舆论情况就是超级头部必死无疑，因为侵蚀了线下零售，吃掉了所有电商的利润，整个平台舆论、商家、政策都在打压超级头部。可以明确的告诉大家，超级头部 IP 的时代在抖音也好，在蝴蝶号也好，都结束了，而且会非常迅速的结束。以后绝不可能有任何的像现在一样有规模有体量，而且还可以无法无天的超级头部了。包括我刘思毅，包括所有在做 IP 的人。无论在每一个品类做得有多大，其实 IP 都不可能是这个领域最牛逼的人。因为这个领域最牛逼的人不可能出来做 IP，他们有太多太多的忌讳了。而且他们拥有足够多的东西了，他们为什么要像我这样录视频、做直播呀？他们不需要这样搞钱了。做 IP 是为了不做 IP，做 IP 是不可能长久的。做 IP 需要不断输出，你心神经气被你完全耗完，要不就是觉得退休了撑不住了。要么就是你飘了被封杀了，要么就是你钱赚够了，你不需要做 IP 了。IP 是一个阶段性的网红，不可能是你一生的事业。所以在做网红的时候就要想清楚你沉淀的是什么？是资源还是钱，还是行业，还是一个资产品牌。网红永远不是资产，一个人格化的东西是不可能成为一个资产的。我自己给我自己的教训就是要做垂直的，以及做 ToB 的，不做 ToC 的 。做私域而不做抖音的，做小众而不做大众的，做高客单价不做低客单价的。以上。-END-想看我的票圈的，扫码加一下，真人运营，随时票圈，和我一起同行。围观暂时免费，抓紧加。
