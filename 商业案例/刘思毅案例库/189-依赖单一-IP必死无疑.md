## 原文全文

依赖单一 IP，必死无疑。 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 这是刘思毅的第1344篇原创持续日更，做特懂流量的创业者博商的曾院长来我们广州的大会直播了，他来分享了他从 2020 年到 2023 年三年的思考。知识付费的王者公司不是任何KOL，不是任何 IP，而是博商。两周前我看了博商曾院长在群响分享的PPT，两周之后我再看还是觉得很牛逼，因为这个 PPT 它完全不对外，我根据我自己的学习，跟大家分享一下博商是如何从 2020 年几个亿到现在 2023 年几十个亿的。博商不是做 IP 的，他们是做 IP 的 MCN ，主打的就是一个铁打的营盘，流水的 IP。一、 IP 必然有生命周期，不能是唯一的压舱石。曾院长说成功的第一点秘诀就是，从来不依赖单一 IP，一个 IP 必然会过气的，周期也就 6 个月， 6 个月之后 IP 就赚不到钱了。因为抖音就是这么残酷，就是封面人物轮流转的。不论你做 IP 还是做 IP 操盘手，你依赖单个网红，必死无疑。所以曾院长说博商要建立一个机制，不断地发掘IP，孵化IP，然后和 IP 顺利的合作，等到 IP 可能过气之后，已经有新的 IP 直接在无缝衔接了。这个模式本质上是博商成功的第一点原因，就是不依赖单 IP 的，它是多个 IP 矩阵，它建立在 IP 的不断地过气，不断地新生，不断地过气，不断地新生这个节奏之上的。二、要赚钱就要有丰富的客单价体系。博商曾院长还说了一个非常非常牛逼的点，就是必须要有自己的高客单价体系。博商的客单价体系是有很多的，365 块钱的线上课，这个是他们自己前端做流量投放用的，不指望前端小课赚钱， 365 就是一个流量费用，自己养团队，分给小朋友。 第二层是 1980 块钱到 2999 块钱的三天两夜的线下课，这是他们的转化课。中间层他们会派很多很多的销转老师，一个桌一个桌的跟着这些老板去转化他们，第二层本来就用来转化的，分给了高客单价的销售，1980 不贵，但是够分了。第三层是 3 万到 10 万的企业微咨询，每一个老师都有这样一个产品，这才是博商赚大钱的产品。第四层是博商和很多大公司 100 万到几百万的咨询，第三层和第四层才是博商母公司，真正赚钱的产品。所以一共有四层产品，每一层产品承担的责任都是不一样的。三、公司成功运转离不开成熟的人才建设体系曾院长最牛逼的就是他的第三点，他的人才体系建设非常的成熟，他的后端分得极其的发达，后端的销售也非常的多，这是是很多抖音公司不具备的，就是他不是一个简单的抖音公司，他是一个商学院公司，只不过在抖音上他又活了，他自己的流量团队是分出去的，他的流量团队是他的合伙人，是他的独立的子公司，而不是博商，博商有 n 个获客，他们有 10 个子公司操盘几百个 IP，这是他们自己投下去的流量的网。大家都知道抖音 IP 成功率就是这么低，所以他用非常非常多子公司，让小弟自己去当老板，别在我博商混着拿钱，自负盈亏。这样就可以摊平抖音流量的不稳定性，所谓的中台就是帮他去卖 1980 的课，然后来销售的人，转到后台去做几万到几百万的课程咨询产品。四、博商已是神话，不可模仿。博商牛逼的点不在于说他抖音做得有多好，而在于他自己的人才机制，本身的商业模式，在一开始就非常非常适合抖音，它适合抖音的客单价高，而且复杂，而且体系化，所以说后端完全可以撑得起前端不赚钱。而大部分像我一样的宝宝，自己做IP，自己做交付，自己做产品自己做流量，啥都自己做。所以你的上限可能就是一年 1000 万，但是博商的上限是一年 30 亿。因为他的 365 块钱可以卖几万份，这几万份都给他的流量操盘手分，流量操盘手疯了，但是这几万份都是精准的客户，再被这几千个销售一起转化，就可以产生几十亿的销售额。曾院长说得非常清楚，他希望能做中国商学院第一，这就是博商的威力，因为抖音成就了博商，也因为博商自己的人才体系和产品体系才有今天的博商。如果我们这些做知识付费的人想复刻博商是百分百是必死无疑的。今天太多人说我要做什么领域的博商，厨师界的博商，月嫂界的博商，管培界的博商，体育界的博商，各种垂直领域博商，他们都没有知道博商的精髓。博商的精髓在于他的商业模式体系，还有人才体系以及流量体系都百分百和抖音匹配，这是博商成功的核心原因。任何人只模仿博商的抖音技术是必死无疑的，因为博商的抖音技术并不稀缺，稀缺的是如此匹配的商业模式～-End-博商为什么可以连续绵延牛逼 3 年，还在一直不断在抖音和视频号战场取得成绩，第一，所有老师都射程范围内，而且每一个老师都是自己的操盘作品，越做越能做好老师，IP 不是培养孵化的，是筛选的。第二，操盘手体系、后续升单变现体系、激励体系以及疯狂抓住红利期的能力，都很强，体系建设是核心能力，没有后续的商业化变现，哪有这么激进的团队成本投入和投放消耗。第三，创始人的胸怀，曾院长还是让更强的新一代操盘手成为了他生态内最强的引擎，而不是让他们自己出去干，这很重要。上次广州大会，很荣幸邀请到博商创始人曾院长分享他的战略和体系，这次广州群响大会，十分荣幸，邀请到博商抖音打法创始人，博商最大操盘公司获客 CEO Becky 返场，重点分享：博商这 3 年的抖音和视频号的 IP 历程、教训、经验、判断，十分期待，想参加的加我报名哈。
