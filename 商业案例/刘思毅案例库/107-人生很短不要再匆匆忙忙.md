# 人生很短，不要再匆匆忙忙。

> 活在当下很难

## 核心要点

- 把 todo list 中的一些东西全部删除
- 不断抛弃、不断选择
- 漂浮在可能性的天空
- 安定于自己选择的那一条路
- 拖延掉所有不重要的事儿

## 原文全文

人生很短，不要再匆匆忙忙。 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 6月6日（周二）“群响四周年”刘思毅携手15位最具代表性的群响会员一起聊聊新时代的流量变迁与个体创业点击预约直播～看了 3 天，认认真真一字一句地看完，从兰州回来的飞机到上海游学结束，这本书又是被心灵鸡汤的封面耽误了。一、作者是一个生产力专栏作家出身，但发现教大家的东西，都是屁话，番茄工作法和任何效率技术问题都是屁话。因为人永远无法完成自己的 todo list，时间有限，事情又多，而且不在自己的控制范围之内，怎么完成。不能全部完成，就是要把 todo list 中的一些东西全部删除，一件一件完成，完成不了就瞎 jb 过，就行了。于是他写了一本反效率主义的人间真相书，核心观点就是，人生有限、活在当下、不必求全、小满即可。而且要不断抛弃、不断选择，只能选一个，接受只能选一个，不要漂浮在可能性的天空，要安定于自己选择的那一条路。每时每刻，都要清楚，自己的选择，尽管遗憾，但因为是唯一的自己的选择，所以真实的缺憾，亦是美丽。二、作者文笔优美，翻译得很好。写的超级好，视角独特，然后推翻了我之前「YOLO」的价值观。他告诉我一个道理。「接受人生的有限性，体验是不会完结的，选择了这个就不能选择那个，因此你要做出承诺，要选择一边，丧失这一边之外的所有选择。你无法把握所有事情，你也无法拥有所有体验，你短暂的一生，只有 4000 周。你选择的任何一个体验在地球上的所有体验中都构不成任何分量。但那是你的选择，真实的、排他的、稳定的选择。因为人生命有限，于是我们要慢、不要急，要拖延掉所有不重要的事儿，不要沉浸于 Todo 列表和优先级排列。而是除了那个最重要的事儿，其他都是垃圾，不用管。」三、太过于震撼，我要摘录一些感受。1，生产力是一个陷阱，高效只会让你更加忙碌。效率太高，容易陷入效率陷阱，效率越高，越有做不完的事情。最有效的策略反而是，到点下班，做不动就跑路。2，时间不是流，时间不是一个账户，时间不是资源，不是抽象的。时间就是我们自己，时间就是生活本身，一定要接受：每一件事儿，都需要消耗本身需要的时间，因为时间就是用来这样消耗的。甚至不能叫消耗，这就是人生本生。3，不要恐惧时间有限。正因为有限，而不是永恒，我们此时此刻做的选择和体验才弥足珍贵，才足够有体验感。4，必须要勇敢直面生命的有限性。工作、生活、爱情、关系都要选择一面，接受不圆满，才能获得当下的平静。5，审慎地做出重大、艰巨、不可撤销的承诺。不要害怕错过恐惧症，错过是一种必然，正是因为错过，我们的选择才有意义。6，不要为了体验太多而让自己疲惫。因为是体验不完的，反而要细致地品味你已经选择的，而不是为了打卡和历程碑。活在当下这个哲学很难，一起学习。服务一群人、记录一群人、代表一群人,这一群人是你们 ,年轻的中国创业宝宝和操盘手宝宝给我一个"在看"！👇与我同行！
