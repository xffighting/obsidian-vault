# 和合伙人撕了一逼，开心。

> 这次是一个很好的课题，创始人的嫉妒心。

## 核心要点

- 讨论群响私域班的课程框架更新
- 离职吗？不对啊
- 在私域班直播
- 到 3 月了
- 是群响会员线的 IP

## 人物盘点

## 原文全文

和合伙人撕了一逼，开心。 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 这是刘思毅的第1652篇原创持续日更，做特懂流量的创业者和合伙人撕了一逼，开心。一、前几天在罗马醒来，国内 15 点，正好要讨论群响私域班的课程框架更新。讨论的过程非常群响，就是我提出议题，武断结论，大家拼命拉我，往回走，我步子大，自己的能量只有 1 倍。恨不得用 10 倍，好在有团队拉住我。我比较享受这样的过程。Toby 当天大陆晚上，罗马下午茶时间给我私信说，要和我聊聊。我说是微信还是电话，他说电话，我心里一紧。一般团队有大事儿才会直接私信电话聊，这和谈恋爱一样，中国人之间始终不擅长坦诚直接沟通。美国人说 We need to talk，真的奔着解决问题去的。我们中国人一般不说我们谈谈，我们的主流文化特点还是比较保守。所以我心里一紧，完犊子，这逼要离职吗？不对啊，没看出来离职的趋势啊，到底是什么呢。于是我和他打电话。二、Toby 比较紧张，跟我复盘了一下上午的开会，我们都觉得这很正常，都是小事。虽然我比较武断冒进，这就是我的风格，然后他说如果你要在私域班直播，那就直播。虽然我不支持，但既然决定，那可以试试，就这么干，可以。我心里一暖，然后等他说正事儿。开始了，他说，自从来杭州就开始写公众号，从十年体，到现在的公众号很多时候日更。然后再到今年过年，我自己也学着拍短视频，我真的有一个心结想跟你说。我本来想等你回来，但是看你回来要到 3 月了，所以现在就想跟你说。我一直很在乎你对我「作为群响合伙人做 IP 的看法」我一直担心你是不是有小九九，是不是会不开心。会觉得 Toby 是不是要单干，会不会影响到刘思毅在群响这个盘子的核心地位。我就想告诉你，如果这个影响我们俩的关系，我宁愿不做。我说我不是给你点过赞吗？而且点了好几次。他说，三次，我一直记得，我很关注你对我的点赞。我知道，所以我才更在意这个事儿，更想说清楚。我不想因为这个事儿不清楚影响我们俩的关系，如此信任、顺滑的合伙人关系。我说，我不仅不反对，还完全支持，还想说，你以后必须要是群响会员线的 IP。而我要奔往下一站，你必须要替代我，我才能继续往前走。不仅是你，Allen 也是如此，你是 IP，Allen 是 CEO，你们向前，其他操盘手才能向前走。三、Toby 说话很颤抖，这需要很大的勇气。一把手通常「卧榻之侧、岂容他人鼾睡」创业 5 年做 IP，风吹日晒这么艰难但为什么这么愿意做。不就是「私我」被滋养和肯定吗？为什么？因为全公司的舞台和资源都给了我。对于表演型人格自恋狂魔，一条朋友圈有 30 万人可能刷的到，当然刺激。但现在，Toby 作为我的合伙人，作为曾经的实习生、小弟，也开始做 IP。而且做得很好，而且有自己的粉丝，而且有自己不断发展、自闭环的逻辑。我怎么看？这是一个很好的课题，创始人的嫉妒心。四、人和人的关系是如此复杂多变，我爱每一个和我一起工作的同事，特别是一起战斗肩并肩的合伙人和操盘手们。他们是群响的火焰，是我的子弟兵，我的兄弟我的团。然而另一方面，是幽暗复杂的人性。核心本质是关于人的主体性权力课题。在存在感这个问题上，成王败寇的直男逻辑 —— 是有你就没有我，有我就没有你。存在感只有这么多，权力让人猜疑和嫉妒。琅琊榜的梁帝就是最典型的例子，越走越窄，众叛亲离。作为老板，一个公司的独裁者一定会有这样的邪恶倾向。Allen 很喜欢反对我，拉着我，在另一面批判我，劝我理性。Toby 很喜欢做 IP，而且热爱表达，有自己的基本盘和表达的战场。我们的操盘手都有自己的心智、风格、偏好、想法和领地意识。这是一个人成为完整的人，一定会有的东西。当你启迪团队成为牛逼的人，就不能不正视自己「主体性」被「侵占」之后这个课题，这是人之为人的特点。而一个想成为教练、成就更大的事儿的老板，会觉察、接纳、穿过它。因为主体性侵占本身是一方面，而另一方面，百花齐开，胜过一花独开。我自从 30 岁后，非常明确地意识到，我不可能一辈子做 IP 这个一线战将，这是我的阶段性工作。我还是想做一个企业家、创业者，我的内心渴求巨大影响力。而一个人承载的影响力太有限了，一个人直接面向观众，不如一群人做好多个舞台，一起面向观众。五、一人盛开，不如集体绽放，过去的刘思毅不理解，现在的刘思毅充分理解。第一，一个人的有限性，在时间上、在情绪波动上、在能量上、在生命力上。于是一个创始人最大的作品是公司，而不是个人 IP。个人 IP 是核心手段，我的是，Toby 的也是，都是公司成长的阶梯。一代一代的 IP 和操盘手的传承和孵化，群响不断向前走。第二，作为创始人，你能否理解，让合伙人和操盘手绽放，你也可更加绽放。公司共享繁荣，养料相互滋养，能量的互相链接让团队的文化更强、更明确的，于是传承继续。Toby 是 IP 型人格，那就是要有舞台，有表达欲释放的空间。Allen 是 CEO 型人格，那就是要有战场，有柴米油盐酱醋茶，让他协调让他操心让他成长。让别人成长，不仅不妨碍我的成长，而且还会让我共享他们的成长带来的增量。无论是公司，还是创始人本身。格局打开后，自己更轻松，更开阔，更加从私我的灌溉，到「神性的自我」的进化。人人皆可为「神」，我亦在途中。谢谢 Toby 作为合伙人的坦诚，谢谢我们彼此给的能量链接。我每天写 10 条朋友圈，关于旅行、案例、搞流量、创业资源。这是我的个人微信号，抓紧扫码加我，然后发一下自我介绍哈👇
