## 原文全文

没有团队的董宇辉，还能火吗？ 原创 爱学习的刘思毅 群响刘老板 在小说阅读器中沉浸阅读 这是刘思毅的第1517篇原创持续日更，做特懂流量的创业者我在香港刚刚吃完董宇辉这个瓜，我真的很想打死董宇辉和他们的那个些真爱粉们。大家不知道一个 IP 背后是有一个巨大操盘团队。因为一个人是根本没有精力做完所有事情的，不是能力问题，是认真的没有精力。如果你要做一个 IP，内容是非常关键的灵魂性的东西， IP 是赋予内容情绪，赋予内容价值观的关键生产要素，这些都没问题。但是绝对不能忽略 IP 背后帮 IP 提供选题、提供素材、提供文案的人。一、以刘思毅为例，看 IP 地板砖的 IP 团队。刘思毅这个 IP 不大不小，一天在抖音上 10 万播放量，地板砖级别的IP。一天在视频号上 10 万播放量，仍然地板砖，一天公众号上1万阅读量，依旧地板砖。即便如此，我背后有 20 个助理，让我分别拆解一下：第一，我有 8 个助理，帮我天天发朋友圈。第二，我有 3 个助理，每天帮我找选题，每天帮我陈列选题，和我一起切磋到底要说哪些内容。第三，其他助理全方位的在帮我剪辑视频，拍摄视频，以及精简我所有抖音视频号上的这些历史素材，以及直播的历史素材。一个 IP 也没有任何立场说你自己的所有的结果都来自于你自己的才华。成功源于大家，这是一个 IP 的一个自律的心态，也是一个基本的道德。我前段时间 30 岁直播的时候，如果不是我兄弟们帮我提供各种 PPT 、各种照片、各种文案以及各种选题框架，能支持 9 个小时的直播吗？9 个小时本身这个直播单场就已然是耗尽我所有生命力的了。我的内容全部来自于我的团队，我的操盘手为我的成功贡献了必不可少的 10% -20%，但这个比重问题不重要。一个 IP 的成功本身，不能把 IP 占多少比例，助理占多少比例严格定义切分。IP 和内容团队亲密无间的合作本身是不可以被否认的。二、强如张琦，背后是一个出版社般庞大的内容中心。我跟博商老师交流，我觉得张琦已经够牛逼了，上天下海无所不能。所有的话题都能说，我就问了博商的人，张琦老师的选题到底是怎么做的？他们非常坦诚地跟我说了几点：第一点，张琦本身是一个有十几年经验的讲师，他自己的商业认知和台前的脱口秀能力可以让他火起来。一周时间就干了差不多 100 万的粉丝，几千万的播放量，这些都没有问题。第二点，但是之后因为自己要不断地做矩阵，不断地满足粉丝对新内容的期待。张琦老师背后是有一个和出版社一样庞大的内容中心的，兄弟们，他有自己的主编，他有自己的 CEO，他有自己非常非常严格意义上的一个内容编码程序内容。每一个 IP 都不能只说，这他妈全是我自己的功劳，你没有你的操盘手，你没有你背后那个人当你的CEO。你啥都不是。三、董宇辉翻车背后，是老板与 IP 必然分手的窘境 。如果你选择做操盘手CEO，你最大的痛苦是发现 IP 永远想跑路。IP自己一旦被孵化出来之后，无论你多让步，让步多少个百分比，还是多么努力跪下来的。IP 一旦成为一个超级大主播，他就一定想跑路，他一定想做自己。圈内有一个非常知名的网红，他原先其实是一个名不见经传的小孩，然后被自己的投资人孵化，投资人很看好这个业务，于是和他成为了合伙人。既是投资人又是合伙人，投入自己的时间和金钱，所以说这个人占大股。而这个小孩占30%，合伙人和投资人占70%。这个小孩一朝成名，拿到了 500 万的粉丝，成为了小股东，所以顺理成章的成为了这个公司的顶梁柱。原先投资人投入的几百万其实在这个小孩面前都是他的束缚，都是他的凭什么和意难平。这个小孩就不想管公司的业务，他沉迷于自己做内容，夯实自己的羽翼，提高自己的影响力。一心一意只想有朝一日自己有足够多的筹码跑路。于是他真的跑路了，但 IP 一旦跑路，带来的必然是两败俱伤。因为投资人和合伙人拿着这个小孩的抖音账号，它可以直接改变这个小孩儿所有的资产的性质。蓝 v 化、改名，以及曝光这个小孩儿所有见不得人的丑闻，所以 IP 就脏了，脏了就很难起势。对公司来说必然是巨大的损失，因为大家关注账号都是冲着 IP 来的。没有了 IP ，流量断崖式暴跌，没有内容就没有观众，所以用户纷纷离开。这个小孩儿，他拥有自己的脸，IP是一个可以动的生命。所以说这个小孩儿自己出去跑路，就可以直接号召一帮自己的铁粉，又做一个自己的号。但是起号何其辛苦，要熬过最难的冷启动期，还要被原来公司不断的泼脏水，不断的打压，不断的经受身体和精神的压力。然后要自己对产品，自己去对商务，开始做原来公司操盘手要解决的一切杂活。这就是一个两败俱伤的结局。人生就是这么遗憾和残缺的，该如何破局呢？欢迎各位老板切磋讨论～-
