大家好，我是爱整活的苍何。

用了 Openclaw 一段时间，说实话，香是真的香。

但有个问题一直在折磨我。回复慢、卡顿、还时不时抽个风，稍微复杂一点的本地任务，等得我脑壳疼。

![[笔记同步助手/images/e06498ad1f523a366c84ae13351133b3_MD5.png]]

本地 Agent 这东西，追求的就是快和稳，要是每次都要傻等，那体验就稀碎了。

我给你们看样东西，这是我用本地 Agent 做的图片排版，一句话自动将多张照片精美重排。

![[笔记同步助手/images/445e1732d82521fb123858f71ac0b4fb_MD5.png]]

整个过程花费不到 5 s，把截图拼接，标注，甚至美化这事，再也不用 PS 了、

![[笔记同步助手/images/93ff8d0cb2a659878fb8c78f25e76f91_MD5.png]]

> 这张图的小人、标注、拼接全部是 Agent 自主完成的，而且速度贼快。

今天元宵节，用上小红书 Skills 写个小红书文案做个图片，自动发到小红书吧。

![[笔记同步助手/images/b743f47da9067505ee6ecaa1426dcdd2_MD5.png]]

这是 Agent 直接唤起浏览器，安全性更高。

我让该本地 Agent 直接搜索最新的 AI 新闻，然后自动帮我发一条 X，含配图，也是轻松拿捏。全程，我就只是用闪电说发了条语音指令，我就喝茶看着。

![[笔记同步助手/images/ead63a6123a161dee7c1697c2d7be904_MD5.png]]

链接我的公众号文章解析技能，丢个链接甩给 Agent，自动提取要点及文章详细信息：

![[笔记同步助手/images/63f55fd7bae6a7c1acc150894402f886_MD5.png]]

然后调用信息图生成 skills，快速生成几张信息图，也可智能合并生成一张。

![[笔记同步助手/images/bf16e7e4d909480c77591cbbb957f73a_MD5.png]]

甚至完全可以再次借助小红书技能，直接转换为小红书文案及图片，并自动发到小红胡。

要知道，我之前用秒哒开发的“公众号一键转小红书”产屁薯图，迭代了 230 次，这下好了，用 Agent 又革了自己的命。

![[笔记同步助手/images/b836e4badf6a859f57e2f13055d911df_MD5.png]]

现在，通通不用，你只需要把任意公众号文章链接丢给这个 Agent，你就能转小红书，甚至，你还能借助 remotion-best-practices skill 生成教学视频。

说到视频，我甚至将之前开发好的漫剧 Skills，配合抖音视频下载 Skills、视频解析 Skills，直接来个爆款视频极限二创及复刻。

比如，这是一个我本地的带货视频：

我现在直接让这个 Agent 分析并写一份详尽的报告：

![[笔记同步助手/images/87c98d71156e3148432b92bb10c97f1f_MD5.png]]

这是 Agent 逐帧解析总结后给的详细报告，从叙事结构到专业营销手法，分析的相当透彻。

![[笔记同步助手/images/759264216fe128c3db7edebf0755412b_MD5.gif]]

基于这份报告可以生成带图片的公众号文章，我给的提示词是这样的：

​

```
基于你的分析，帮我生成一篇公众号文章，要有短标题，要有配图，配图你可以从视频中去拿，尽量口语化，句子和段落不要太长，帮我生成 markdown 文章
```

![[笔记同步助手/images/06381ac27a84655f68c4d8a190c2fab4_MD5.png]]

这还没完，基于解析的结果，我直接配合之前我的漫剧 Skills，仿照着该视频把 AI 眼睛这个产品，做成一个古风版本的带货视频。

![[笔记同步助手/images/c4d6ccd4841b66975bb3c97b3e4eb7b7_MD5.png]]

该 Agent 会调用我的 manga-style-video Skills，根据分析报告中的爆款手法，自动写脚本，自动生成分镜视频，并完全保持角色和场景的一致性。

现在这个过程全部由 Agent 自主执行，我的 Skillls 负责生成视频及控制风格。

> 虽然还有瑕疵，比如前半部分没旁白，只有字幕，主要是我一开始的提示词没做控制。

但已经很不错了，要知道，我只给了一个参考视频而已，全凭 Agent 自主干货。

当然，如果你是看到的抖音上的爆款视频，你想复刻，你只需要在这个 Agent 安装一个抖音下载 Skills，以及我的长视频分析 Skills。

原视频在这里：

![[笔记同步助手/images/3ea958d2af3e883e205dee97ce10f418_MD5.png]]

然后只需要把他丢给 Agent，你就去喝茶就行了。

![[笔记同步助手/images/423aa8989453d83d210cf3d347892b28_MD5.png]]

视频下载好后，解析视频，这个原理是借助大模型能力，无需单帧截图解析。

![[笔记同步助手/images/85079022e37fc905368ca8ad4c0fe966_MD5.png]]

然后调用漫剧技能，转为吉卜力风格的漫剧视频。

![[笔记同步助手/images/924a478630f839180ff9a432bc6de9ae_MD5.png]]

视频就复刻好了：

虽说也没那么完美，但至少整个流程是完全通的，接下来只需要做优化就好了。

短视频创作的门槛又再一次被这个 Agent 给拉低了。

聪明的人估计想到了，用这东西，去做视频，切片，配合剪辑 Skills，卧槽，简直无敌啊。

由于时间关系，这里我就没再配了。

​

> btw，所有本文用到的 Skills 我都已经上传到了我的 Skills 仓库，感兴趣的可以私信公众号回复：canghe-skill

![[笔记同步助手/images/2f930088b4d56a4215018d73e29b140f_MD5.png]]

卖了这么长的关子，估计你很好奇这是个啥 Agent 吧？

这个就是阿里推出的桌面端 Agent，叫 QoderWork，之前也给大家分享过，很厉害，但之前只支持 Mac，现在也支持 Windows 了。

![[笔记同步助手/images/a9e3eb33352a41ea94bc05ce244a7d9f_MD5.png]]

和 Openclaw 不同的是，QoderWork 不需要部署，本地电脑就可以直接使用。

只需通过简单的自然语言，就能调用你电脑里的应用，帮你完成复杂任务，比如⽂件整理、数据处理、文档⽣成，自动做 PPT 和视频等任务。

以上所有的任务都是在 QoderWork 中完成的。

![[笔记同步助手/images/2adc9d29cccf4b9511d7272bb1cdb1e5_MD5.png]]

我看 QoderWork 还增加了标准与旗舰两档模型可选择，⽇常⽂档处理、信息整理选就用标准模式就够了，省 token，复杂任务就用旗舰模型。

![[笔记同步助手/images/4c2332e866f479de9f64660c7a76e722_MD5.png]]

QoderWork 是运行本地的沙盒环境中，在首次打开就会启动这个环境，相对来说，会更独立安全和为嗯定些。

我观察了下这次最大的不同是，QoderWork 集成了很多场景化的 Skills，比如我用到的小红书图片 Skill 及文章信息图 Skill 都是在市场一键安装的，不用任何配置。

![[笔记同步助手/images/7a8822d35a2fea59f2f4486af6903054_MD5.png]]

像我自己的 Skills 也可以手动安装，或者让 QoderWork 自动给你安装，比较方便。

![[笔记同步助手/images/994d0a7a98d9698898cfe41ced6cc28b_MD5.png]]

相比于 Openclaw，我的感受比较明显，一个是更稳定了，另一个就是对于很多任务会更快。

对于本地电脑办公类任务，你完全可以交给 QoderWork。

它甚至能帮忙识别发票，然后整理金额放到 excel 表格中，计算报销的金额，这玩意，谁用知道有多爽了。

![[笔记同步助手/images/597b09c2f6d38d4abe52f2a73f7414e5_MD5.png]]

当然一句话合并多张发票到一个 PDF 里面，也非常简单方便，在 QoderWork 中都能完成。

![[笔记同步助手/images/bcf99181172dffea392d60e086e2853d_MD5.png]]

说真的，用完 QoderWork，我有个很强烈的感受。

以后的内容创作，拼的不再是你有多会剪辑、多会写文案、多会 P 图。

**「拼的是你会不会用 Agent，会不会组合 Skills」**。

视频复刻、一键转漫剧、公众号转小红书、发票整理，以前这些东西哪个不得整个把小时？

现在丢给 Agent，你去喝茶。

短视频、图文、视频创作的门槛，又被拉低了一个台阶。

而且这才是开始。

我现在已经越来越相信一件事：**「下一个时代，人和人之间最大的差距，就是会不会用 AI 帮自己干活。」**

你现在不上车，以后别说没人提醒过你。

感兴趣的可以去试试 QoderWork。

觉得有用的话，点个赞吧，让更多人看到。
