## 核心观点

### 人脉的核心思维

**神经元模型**：把自己当做神经元，四周的突触是你的人脉，许多神经末梢是你的扩展人脉。

> "善于经营人的人，才能让人成为你的资本和资源。投资'人'比投资任何行业都划算得多。"

> "得麒麟才子者得天下" —— 信息就是权力

### 人脉建立公式

**人脉 = 人情 + 手段**
- 人情是基础
- 手段是方法

### 反向推导思维（目标倒推法）

```
1. 如何掌控订单？ → 掌控买家
2. 如何掌控买家？ → 买家相信你，制约他，不可取代
3. 如何做到？ → 手里有足够的资源
4. 怎么获得？ → 无所不能的网络
5. How? → 社交软件、翻墙软件、开发信
6. 开发信没用？ → 那是你笨
7. 怎么变聪明？
   - 广撒网：找到对你感兴趣的人
   - 精准定位：找到对产品感兴趣的人
   - 心理分析：让他们欲罢不能
```
